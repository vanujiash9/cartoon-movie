📚 Tích hợp vào FrontEnd thật

1. Include shared API file:

<script src="./js/shared-api.js"></script>

2. Add share buttons to movie pages:

<button onclick="shareMovie(movieId, 'facebook')">Share Facebook</button>

3. Update achievements page:

const achievements = await MaxionAPI.getUserAchievements();
